# 斯坦福 vibe coding 课程

## 核心内容

- 课程强调真实工程场景下的人机协同，不止是 demo。
- 覆盖 prompt、IDE、terminal、agent 管理等能力。
- 目标是显著提升开发生产力与交付效率。
- 提供课程视频与 PPT 入口：themodernsoftware.dev。

## Bubble 的洞察

- 这门课价值在“工程工作流”而不是单点提示技巧。
- 将 agent 管理纳入课程，说明已进入系统化阶段。
- 对学习者而言，最关键的是把课程内容映射到自身项目。
- 若配作业制实践，吸收速度会显著高于纯观看。
- 可作为“AI开发训练营”的课程骨架模板。
