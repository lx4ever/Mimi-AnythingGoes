# UI/UX 相关 Skill 清单（8项）

## 内容

1. ui-skills / ui-skills

• 15个独立 UI Skills 集合
• baseline-ui（Tailwind 标准）
• baseline-react（React 组件）
• baseline-vue（Vue组件）
• minimalist-ui（极简 Notion/Linear）
• soft-skill（高端视觉）
• 覆盖主流前端框架和设计风格，一键配置即用

2. pbakaus / impeccable

• 增强版 frontend-design skill
• 20个设计命令：
/polish 精修界面
/audit 设计审查
/typeset 排版优化
/arrange 布局调整
• 覆盖排版、色彩、空间设计、动效、交互、响应式、UX写作等，减少“AI味儿”

3. Leonxlnx / taste-skill

• UI审美评分 Skill
• 从排版、色彩、对比度、间距、字体层级等维度评估
• 目标：拒绝 boring/generic/slop，适合和其他 UI 升级 skill 搭配

4. anthropics / agent-skills

• Anthropic 官方 Agent Skills 仓库
• 示例：frontend-design、creative-design、tech-writing
• 还有文档处理能力：docx 解析、pdf 提取、pptx 生成、xlsx 分析

5. nextlevelbuilder / ui-ux-pro-max-skill

• 全栈 UI UX 设计 AI Skill
• 50+ 设计风格、97 配色方案、9 种技术栈
• 覆盖 React/Vue/Angular/Svelte/HTML/CSS，一站式生成专业 UI

6. better-auth / better-icons

• 智能图标推荐 Skill + MCP Server
• 基于 Lucide、Phosphor、Heroicons 等图标集
• 按 UI 场景与语义自动推荐并生成图标代码

7. kylezantos / design-engineer-auditor

• 自动审查设计代码质量
• 检查组件完整性、可访问性(a11y)、色彩对比度、响应式布局、命名规范等
• 提供修复建议和代码示例（像设计工程师审查 Agent）

8. motion-dev / ai-kit

• 动效设计 Skills（motion.dev AI Kit）
• 定义交互动效、过渡、微交互原则、贝塞尔参数
• 提供 Motion Data 与代码实现；Framer Motion 官方出品

## Bubble 的洞察

- 这份清单已经覆盖了完整设计链路：**生成（baseline/pro-max）→ 质检（auditor/taste）→ 精修（impeccable）→ 细节增强（icons/motion）**，结构非常强。
- 你不是在找“一个万能skill”，而是在搭建可组合工具栈，这能显著降低产出风格随机性。
- 你特别强调“减少AI味儿”和“拒绝generic”，说明你的核心目标不是速度，而是可交付质量与审美辨识度。
- 这8项里最有杠杆的是“评分+审查”层：先有审美评分与工程审查，后续所有生成类技能都会变得更稳定。
- 可执行路线很清晰：先定一个 baseline 产出，再叠加 `/audit` + taste 评分闭环，最后用 motion/icons 做增益，能快速形成团队级标准流程。
