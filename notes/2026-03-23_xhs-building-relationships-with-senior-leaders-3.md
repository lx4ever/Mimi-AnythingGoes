# 深度提取｜海外职场：和senior leaders怎么建立关系（3）

链接：http://xhslink.com/o/37VaaeKi2u6

## 1) 证据锚点

- 主题聚焦“如何和senior leaders建立1:1关系”。
- 文案明确三步：发起1:1、提高1:1质量、会后follow-up。
- 方法细节：提前24h发pre-read；议程分为求助/汇报/提供帮助；汇报讲impact不讲细枝末节；请求要具体可执行。

## 2) 底层机制拆解

- 高层时间稀缺，关系建立靠“低认知负担+高执行清晰度”。
- 影响力沟通本质是“把你做的事转成业务影响”。

## 3) 结构模型 / 方法框架

- 1:1模板：目的定义 -> pre-read -> impact汇报 -> 具体请求 -> offer help -> follow-up
- 请求句式升级：从“帮我建立关系”改为“xx leader最关心什么/好恶是什么”。

## 4) 边界与反例

- 边界：适合跨层协作和成长型职场环境。
- 反例：若直属领导高度防御、组织政治强，直接skip-level需先评估风险和节奏。

## 5) 一句话结论

与senior leader建立关系的关键不是“多见面”，而是每次互动都可被快速消费、快速执行、可持续跟进。
