# 深度提取｜每周用AI做一个网站，直到赚一百万，第2个

链接：http://xhslink.com/o/4OMrpuuArGM

## 1) 证据锚点

- 标题与主张：每周至少发一个AI网站，目标赚到100万。
- 本期产品：上班摸鱼健身网站（工位健身/冥想）。
- OCR补充：图中明确写了“2.上班摸鱼健身网站”“以动画引导用户在工位健身、冥想”，并展示功能页（眼部、肩颈、提肛、综合、呼吸冥想、扔掉烦恼）。
- 互动：赞349 / 藏343 / 评36；评论集中问“用什么AI”“网站链接”“第一站赚了多少”。

## 2) 底层机制拆解

- 用“公开连载 + 低成本上线 + 明确场景痛点”构建持续关注。
- 痛点选得轻但高频（上班疲劳/肩颈/情绪），利于传播。
- 但“流量≠收入”，目前更像MVP验证叙事。

## 3) 结构模型 / 方法框架

- 模式：周更节奏 -> 选高频微痛点 -> AI快速开发 -> 发布复盘 -> 下一轮迭代
- 产品模板：2分钟可执行动作 + 低打扰 + 可偷用（工位场景）

## 4) 边界与反例

- 边界：内容热度高，不代表商业闭环已跑通。
- 反例：若无分发/转化机制，连续做网站也可能“有曝光无收入”。

## 5) 一句话结论

这是一个“AI+连续创业公开实验”的第二站，产品方向对，但关键门槛在后续变现系统而非开发速度。
