# Greg Isenberg 的 Vibe Marketing：一人顶一个团队

## 内容提取（OCR整理）

### 核心定义（What）

- 传统营销：10人团队、6周时间、较高预算
- Vibe Marketing：1人 + AI，3天出结果，成本显著下降（文中提到约 90%）

作者将其理解为：
- 从 Vibe Coding 延展到 Vibe Marketing
- 通过 AI 工具把文案、设计、投放、触达流程压缩

### 为什么现在可行（Why）

文中提到三点变化：
1. 工具能力增强（文案/设计/投放一体）
2. 零代码自动化可跑通完整流程（如 Zapier/Make）
3. 成本显著下降，个人可承担

### 三个核心招数（How）

1. 锁定一个“北极星指标”
- 不追虚荣指标，盯一个可验证增长目标
- 90天复盘，不行就调

2. 从 X 回复开始
- 找目标领域头部账号
- 每天做高价值互动（有洞察，而非浅层点赞）
- 逐步过渡到私信触达与反馈

3. 用 AI 自动跑竞品与触达
- 抓取竞品数据 + AI分析 + 生成个性化触达消息
- 目标：从“找人”到“写消息”形成流水线

### 理想的 24 小时发布流程（示例）

- 上午：生成 landing page 并部署（文中提到 Vercel）
- 下午：生成多版本广告文案，抓取目标用户
- 晚上：设置自动化邮件并开始 A/B 测试

### 推荐工具（按图中表格）

- 文案生成：ChatGPT / Claude
- 数据抓取：PhantomBuster
- 自动化：Zapier / Make
- 触达用户：Lindy + AI
- 快速建站：Replit / Claude Code / Vercel

### 文中结论

- 一人公司在 AI 时代的杠杆显著提升
- 差异化在于工具掌握与行动速度
