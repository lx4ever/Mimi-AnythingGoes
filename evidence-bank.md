# Evidence Bank

Purpose: when self-doubt appears, retrieve proof, not panic.

Use this page to store hard things Mimi has already done — so confidence can be evidence-based.

## Quick Reset Script (30s)

当我开始怀疑自己时：
1. 先看最近 3 条“我做成过”的证据
2. 问：我当时用了什么策略？
3. 今天只复用一个最小策略

Anchor sentence:
> 我不需要先感觉完美才开始。我已经有证据，我做得到。
