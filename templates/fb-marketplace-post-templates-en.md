# Facebook Marketplace English Post Templates (Graduation / Move-out)

Use this as copy-paste templates. Replace [brackets].

---

## 1) High-Value Furniture (Desk / Chair / Bed Frame / Dresser)

**Title**
Moving Sale – [Brand] [Item Name] – Great Condition – Pickup in [Area]

**Price**
$[price] (reasonable offers welcome)

**Category/Condition**
[Category] | [Like New / Good / Fair]

**Description**
Hi! I’m selling my [brand + item] because I’m moving out of [city/area].

- Original price: $[original price]
- Asking price: $[asking price]
- Condition: [X years/months], [any minor wear], from a smoke-free / pet-free home [if true]
- Why it’s good: [comfort / sturdy / space-saving / ergonomic]
- Dimensions: [L x W x H]

Included: [accessories / screws / manual / cover]

Pickup only in [exact area/intersection].
Available pickup windows: [time slots].
Payment: Cash / Zelle / Venmo.

If the post is up, it’s available.
Please message with your earliest pickup time.

## Quick Fill Checklist Before Posting

- [ ] Honest flaws listed
- [ ] Original + asking price included
- [ ] Pickup location + time window included
- [ ] Payment methods included
- [ ] "If post is up, it’s available" line included
- [ ] One clear CTA: "Send earliest pickup time"
